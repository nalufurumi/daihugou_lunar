# 大富豪 (Daifugo)

大富豪のゲームロジックをJavaScript(ESモジュール)で実装したプロジェクトです。

## ファイル構成

- `card.js` — カードのデータ構造、山札生成、強さ判定、役の判定ロジック
- `game.js` — ゲーム状態の管理、カードを出す・パスする処理
- `test.js` — 動作確認用の簡易スクリプト (`npm test` で実行)

## セットアップ

```bash
npm install
npm test
```

## 実装済みルール

### 基本

- 1枚出し / ペア出し(同数字2枚以上) / 階段出し(同スートで3枚以上連番)
- ジョーカーは任意のカード・階段の穴埋めとして使用可能
- 任意人数のプレイヤーに対応

### 縛り

- **階段縛り**: 場と同じ枚数の組が連番で出されると発動し、以降は「場の数字+1」の組しか出せない
- **記号縛り**: 直前の組と共通するスートが1つでもあると発動し、以降はそのスートを含む組しか出せない

### 特殊効果 (数字役)

| 数字 | 効果 |
| --- | --- |
| 3 | スペードの3は、場がジョーカー1枚だけの時に限り例外的に出せる |
| 3×3枚 | 砂嵐・ろくろ首。場の枚数・種類を問わず出せる。出した後は砂嵐・ろくろ首でしか返せない |
| 5 | 出した枚数分プレイヤーをスキップ |
| 7 | 出した枚数分、好きなカードを次のプレイヤーに渡す |
| 8 | 場を流し、出した本人の手番が続く(8切り) |
| 9 | 出した枚数が奇数なら、手番の進む向きが反転する |
| 10 | 出した枚数分、好きなカードを手札から捨てる |
| 11 | 「そのまま」「バック」「ステイ」を選択。バックは場が流れるまで強さ逆転、ステイは場が流れるまでJしか出せない |
| 12 | 出した枚数分、好きな数字を指定し、全プレイヤーがその数字のカードを捨てる |
| 4枚同時出し | 革命。ゲーム終了まで強さの序列が反転する(もう一度4枚出しで革命返し) |

## 使用例

```javascript
import { createDeck, shuffleDeck, dealCards, sortHand } from './card.js';
import { createGameState, playGroup, passTurn } from './game.js';

const hands = dealCards(shuffleDeck(createDeck()), 2).map(sortHand);
const state = createGameState(hands);

// 1枚出す
playGroup(state, 0, [hands[0][0]]);

// 7を出して次のプレイヤーにカードを渡す場合
// playGroup(state, 0, sevenCards, { giveCards: [someCard] });

// パスする
passTurn(state);
```
